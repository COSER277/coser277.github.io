写下这篇文章 也属于真正第一篇博客文章 记录与分享给大家

1 介绍

Hexo 是一个静态页面Blog吧 理解为是基于 node.js 制作的一个博客工具，不是我们理解的一个开源的博客系统。其中的差别，比如没什么后台管理呀等等，不过很有意思。
Hexo 通过 markdown 编写的文章，然后 hexo 帮我们生成静态的 html 页面，然后，将生成的 html 上传到我们的服务器(比如 GitHub 上嘻嘻)。
Hexo 是个静态页面生成、上传的工具。

2 安装

1 必需 Nodejs 环境
2 npm install -g hexo-cli (Hexo 脚手架)或 用 cnpm
3 初始化项目 hexo init 项目名
4 进入项目 cd 项目名
5 安装依赖包 npm install 或 cnpm install 或 yarn (我喜欢用 yarn 嘻嘻)
6 跑起项目 hexo server 或 hexo s
7

2-1 本地跑起来测试

1 进入项目目录
2 打开 cmd 窗口 或 power shell
3 输入 hexo s 或 hexo serve
4 打开浏览器 http:localhost:4000

3 目录结构

1 node_modules 依赖包
2 public hexo 生成的公共文件（访问 github 地址时的文件）
3 scaffolds
4 source 资源文件 如 posts 文章（.md 文件）
5 themes 主题存放文件夹
