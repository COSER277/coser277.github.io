再次写下这篇文章，记录经验，与分享给大家

1 介绍

Hexo 是一个静态页面Blog吧 理解为是基于 node.js 制作的一个博客工具，不是我们理解的一个开源的博客系统。其中的差别，比如没什么后台管理呀等等，不过很有意思。
Hexo 通过 markdown 编写的文章，然后 hexo 帮我们生成静态的 html 页面，然后，将生成的 html 上传到我们的服务器(比如 GitHub 上嘻嘻)。
Hexo 是个静态页面生成、上传的工具。
