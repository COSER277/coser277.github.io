1介绍

** Hexo是一个 Blog 理解为是基于node.js制作的一个博客工具，不是我们理解的一个开源的博客系统。其中的差别，比如没什么后台管理呀等待，不过很有意思。
Hexo通过markdown编写的文章，然后hexo帮我们生成静态的html页面，然后，将生成的html上传到我们的服务器(比如GitHub上嘻嘻)。
Hexo是个静态页面生成、上传的工具。
**

2安装

** 1 必需 Nodejs环境
2 npm install -g hexo-cli (Hexo脚手架)或 用 cnpm
3 初始化项目 hexo init 项目名
4 进入项目 cd 项目名
5 安装依赖包 npm install 或 cnpm install 或 yarn (我喜欢用yarn 嘻嘻)
6 跑起项目 hexo server 或 hexo s
