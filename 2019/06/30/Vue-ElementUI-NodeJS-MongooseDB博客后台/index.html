安装教程

  1. 解压压缩包 进入目录
  2. 打开命令窗口 （shift +右键 ）
  3. 输入: yarn install 安装项目所需的依赖包
  4. 等待安装完成
  5. 输入： yarn serve 运行项目
    6。打开浏览器 http://localhost:8081/ （ 根据提示打开）
